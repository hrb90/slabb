# API Engpoints

## HTML API

### Root

- `GET /` - loads web app

## JSON API

### Users

- `POST api/users`
- `GET api/users`

### Channels

- `POST api/channels`
- `GET api/channels`
- `GET api/channels/:id`
- `PATCH api/channels/:id`
- `POST api/channels/:id/subscribe`
- `POST api/channels/:id/messages`

### Messages
- `PATCH api/messages/:id`
